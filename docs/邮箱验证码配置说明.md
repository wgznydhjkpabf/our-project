# 邮箱验证码配置说明

## 1. 执行数据库迁移

已有数据库请执行：

```cmd
mysql -u root -p campus_trade < sql/email_verify_migration.sql
```

新安装直接执行 `sql/init.sql` 即可。

## 2. 开发模式（不配置 SMTP）

默认 `app.mail.enabled: false`，验证码会打印在后端控制台：

```
========== 邮件未配置，验证码（开发模式）==========
收件邮箱: xxx@163.com
验证码: 123456
```

## 3. 生产模式（真实发邮件）

编辑 `backend/src/main/resources/application.yml`：

```yaml
spring:
  mail:
    host: smtp.163.com
    port: 465
    username: 你的邮箱@163.com
    password: SMTP授权码（不是登录密码）
    properties:
      mail:
        smtp:
          auth: true
          ssl:
            enable: true

app:
  mail:
    enabled: true
```

常用 SMTP：
- 163：`smtp.163.com:465`
- QQ：`smtp.qq.com:465`

## 4. API 接口

| 接口 | 权限 | 说明 |
|------|------|------|
| POST `/api/auth/email/send-code` | 公开 | 注册发验证码 `{ email, scene: "register" }` |
| POST `/api/user/email/send-code` | 登录 | 绑定发验证码 `{ email, scene: "bind" }` |
| POST `/api/user/email/bind` | 登录 | 绑定邮箱 `{ email, code }` |
| POST `/api/auth/register` | 公开 | 注册需带 `verifyCode` |

## 5. 功能说明

- 验证码 6 位数字，5 分钟有效
- 同一邮箱 1 分钟内最多发 3 次
- 注册必须验证邮箱
- 个人中心可绑定/更换邮箱
